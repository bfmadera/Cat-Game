# Gatinho voltando para casa

A simple 2D platformer game made in **C#** using **Raylib-cs**.

In this game, you control a cat called **Gato**, who must cross platforms, collect fish, avoid dangerous red barriers, activate checkpoints, and finally reach home.

## Features

This project was designed to include the following mechanics:

- 2D platformer movement
- Camera following the player
- Double jump
- Run button
- Horizontal dash
- Checkpoints
- Pick-ups (fish)
- Health system with damage
- Credits screen

## Controls

- **Enter**: Start game
- **A / D** or **Left / Right Arrow**: Move
- **Ctrl**: Run
- **Space** or **Up Arrow**: Jump / Double Jump
- **Shift** or **X**: Horizontal Dash
- **C**: Open credits
- **Esc**: Return from credits
- **R**: Return to menu after win or game over

## Objective

Help Gato get back home by:

- jumping across platforms
- collecting fish
- avoiding hazards
- using checkpoints to save progress
- reaching the house at the end of the level

## Technologies Used

- **C#**
- **.NET 8**
- **Raylib-cs**

## Project Structure

- `Program.cs` → application entry point
- `InterfaceCat.cs` → main game logic
- `Game.csproj` → project configuration
